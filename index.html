    Created a small applet that displays to-do list items that need to be done using HTML, CSS and JavaScript.

    How It Works
    First of all, I coded the input bar to always be in focus therefore, whoever is using the list does not need to click the input bar and there is also an event listener on the submit button that allows the user to press the return key and have the to-do item posted in the list. There are two things that can be done with an item, either recycle it(or delete) or mark it as done.
